## Deployment

Designed for **GitHub Pages**:

1. Go to repo Settings → Pages
2. Set source to `main` branch, `/ (root)`
3. Live at `https://<username>.github.io/<repo>` or custom domain

For custom domain (`loriumarchive.com`):
- Add a `CNAME` file to the root with your domain
- Point DNS A records to GitHub Pages IPs
